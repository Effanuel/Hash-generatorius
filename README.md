## Prerequisites
1. GNU C++ Compiler

## Installation
1. [Atsisiūskite naujausią versiją.]()
2. Extract failus.
3. Per command-line:
```
cd HashGenerator
make run
```
4. Executable yra **./HashGenerator/bin/project.exe**
```
cd bin
./project
```
